# 🌌 US2SE Bridge — Live Sync Universe Sandbox 2 → SpaceEngine

Скрипт-мост для мгновенной передачи планет и систем из Universe Sandbox 2 в SpaceEngine. Настрой в Universe Sandbox — летай в SpaceEngine в реальном времени!

## 🚀 Как пользоваться

1.  **Настрой `config.ini`**: Создай копию файла `config.ini.example`, переименуй её в `config.ini` и укажи правильные пути к папкам US2 и SE.
2.  **Запуск**: Нажми правой кнопкой на **`START.bat`** и выбери **«Запуск от имени администратора»**.
3.  **Синхронизация**: Каждый раз, когда ты нажимаешь `Ctrl+S` (сохранить) в Universe Sandbox, скрипт автоматически обновит каталог SpaceEngine и перезапустит его.

> [!IMPORTANT]
> **Всегда запускай батники от админа!** Без этого Windows не позволит скрипту перезапускать SpaceEngine и US2 корректно.

---

## ✨ Основные функции

*   **⚡ Live Sync**: Обновление мира происходит через секунду после нажатия «Сохранить» в US2.
*   **🪐 Умный конвертер**:
    *   **Двухфазная запись**: Сначала записываются планеты, затем спутники (исправляет проблему «пропадающих» лун).
    *   **Повышение ранга**: Если спутник (Moon) улетает от планеты к звезде, он автоматически становится полноценной планетой (Planet), чтобы SE не удалил его.
    *   **Авто-класс**: Скрипт распознает объекты > 20к км и назначает им класс `GasGiant`, чтобы они выглядели как газовые гиганты.
    *   **Климат**: Передает `Albedo` и `SurfaceTemperature` напрямую из симуляции.
*   **🛠️ Тонкая настройка**: Поддержка кастомных звезд через параметр `se_star_name` в конфиге.

---

## 🛠️ Файлы проекта

*   **`START.bat`**: Главный пусковой файл (US2 + SE + Sync).
*   **`STOP.bat`**: Чистое закрытие всех процессов моста.
*   **`us2se_converter.py`**: Движок генерации `.sc` каталогов.
*   **`us2se_sync.py`**: Следит за изменениями файлов симуляции.
*   **`config.ini`**: Твои пути и настройки (в репозиторий не выкладывается).

---

## 🧪 Ограничения (что не попадет в SE)

Фрагменты столкновений ("Дебри"), мелкая пыль и безымянные объекты автоматически фильтруются, чтобы не засорять SpaceEngine.
*   **Минимум для планеты**: 100 км.
*   **Минимум для астероида (sso)**: 0.1 км.
