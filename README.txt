БЫСТРЫЙ СТАРТ
─────────────────────────────────────────────────────────────

  1. Настрой пути в config.ini (один раз):
       us2_simulations — папка с сохранениями Universe Sandbox
       se_install      — папка где установлен SpaceEngine
       se_exe          — полный путь к SpaceEngine.exe
       us2_exe         — полный путь к исполняемому файлу US2

  2. ПКМ на START.bat → «Запуск от имени администратора»
       Батник запустит Universe Sandbox, SpaceEngine
       и скрипт конвертации автоматически.

  3. Сохрани симуляцию в Universe Sandbox (Ctrl+S).
       Скрипт обнаружит новый файл сохранения, сконвертирует
       его в формат каталога SE и обновит игру.
       ВАЖНО: Это не стрим! Обновление только по нажатию Save.

  4. Для остановки: ПКМ на STOP.bat → «Запуск от имени администратора»
       Закроет все связанные процессы.

─────────────────────────────────────────────────────────────
  ! ВАЖНО: БАТНИКИ ЗАПУСКАТЬ ОТ ИМЕНИ АДМИНИСТРАТОРА !
  
  Без прав администратора скрипт не сможет корректно
  управлять процессами SpaceEngine и Universe Sandbox.
─────────────────────────────────────────────────────────────

ФАЙЛЫ ПРОЕКТА
─────────────────────────────────────────────────────────────

  START.bat          — запуск всей цепочки (US2 + SE + Sync)
  STOP.bat           — остановка всех процессов
  config.ini         — все настройки путей и параметров
  us2se_converter.py — движок конвертации .ubox → .sc
  us2se_sync.py      — следит за сохранениями и запускает конвертер

ФИЛЬТРЫ (что попадает в SpaceEngine)
─────────────────────────────────────────────────────────────

  ✓ Планеты с радиусом > 100 км и именем (не "Фрагмент")
  ✓ Спутники планет (Moon)
  ✓ Астероиды (Small Body) с любым размером > 0.1 км
  ✗ Фрагменты, пыль и мелкие обломки
  ✗ Безымянные объекты

ПОИСК В SPACEENGINE
─────────────────────────────────────────────────────────────

  Нажми Enter → введи название планеты из симуляции
  Система обычно привязана к центральной звезде.
